## 실행 방법 / 트리거
`대시보드_실행방법.md` 기준: ① `server/index.js`를 node로 실행(로컬 API 서버 :3001) ② `npm run dev`(Vite :5173) ③ 브라우저 접속. 배포판은 Vercel 서버리스 함수(`api/*.js`)로 프론트+백엔드 단일 배포.

## 데이터 흐름
구글시트(`1cqLifj...`, 자사몰/외부몰/불량 교환, 재고관리 등) → Express(`server/index.js`) 또는 Vercel API가 google-spreadsheet로 조회·업데이트 → React 프론트(자사몰교환/외부몰교환/재고부족/미처리건/리드타임/실적/불량분석 탭) → 지연건은 `send_single_python.py`/`alimtalk_core.py`(로컬) 또는 `api/_alimtalk.js`(배포)로 루나소프트 알림톡 발송

## 기술 스택
React 19 + TypeScript + Vite + Tailwind + Recharts, Express + google-spreadsheet, Vercel 서버리스, Python(알림톡 발송 원본 로직)
