# 교환 대시보드 직접 실행하는 방법

대시보드가 꺼졌을 때 이 파일을 보고 따라 하시면 됩니다.

## 1. 터미널(Terminal) 앱 실행

## 2. 서버(Server) 켜기 (데이터 연결)
1. 터미널 하나를 열고 아래 명령어를 복사해서 붙여넣고 엔터(Enter)를 칩니다.
```bash
cd "/Users/deepdive/Documents/강희/Github/교환대시보드/cx-exchange-dashboard/server" && /usr/local/bin/node index.js
```
2. `Server running at http://localhost:3001` 라는 메시지가 나오면 성공입니다.

## 3. 대시보드 화면(Frontend) 켜기
1. 터미널의 **새 탭**(Command + T)을 하나 더 엽니다. (위의 서버는 계속 켜져 있어야 합니다.)
2. 아래 명령어를 복사해서 붙여넣고 엔터(Enter)를 칩니다.
```bash
cd "/Users/deepdive/Documents/강희/Github/교환대시보드/cx-exchange-dashboard" && /usr/local/bin/npm run dev
```
3. `Local: http://localhost:5173/` 이라는 주소가 나타나면 성공입니다.

## 4. 확인하기
브라우저(Chrome 등) 주소창에 `http://localhost:5173/` 을 입력하고 접속하세요.

---

### ⚠️ 에러 발생 시
- **"node not found"** 또는 **"npm not found"** 에러가 나면 명령어 앞에 `/usr/local/bin/`을 붙였는지 확인해 주세요.
- 제가 현재 배경에서 이 명령어들을 실행해 두었으니, 지금은 바로 [http://localhost:5173/](http://localhost:5173/) 에 들어가시면 됩니다.
